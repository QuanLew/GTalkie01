### To run frontend + voiceRegconize
in root, execute: <br />
`npm install` <br />
`npx expo prebuild -p ios` <br />
`npm run ios` <br />

### To run frontend
in root, execute: <br />
`npm install`<br />
`npm start`<br />
then choose `i`<br />

### To run backend (for sending emails only)
in a new terminal, navigate to /backend<br />
execute<br />
`npm install nodemon`<br />
`npm run dev`<br />
